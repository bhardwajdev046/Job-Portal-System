# 🚀 HireHub - Modern Job Portal

A premium Job Portal built with **Django 6.0**, **Tailwind CSS**, and **AOS Animations**.

### ✨ Features:
- **Role-based Auth:** Separate dashboards for Employers and Candidates.
- **Job Management:** Post, edit, and track job applications.
- **Email Notifications:** Real-time HTML email updates on application status.
- **Modern UI:** Dark/Light mode support with smooth scroll animations.

### 🛠️ Tech Stack:
- **Backend:** Python, Django
- **Frontend:** Tailwind CSS, AOS (Animate On Scroll)
- **Database:** SQLite (Development)
- **Communication:** SMTP Gmail Integration
